# Movie Search App

This project was built while learning **JavaScript** and working with APIs.

It is a movie search application that fetches movie data from the **OMDb API** and displays details similar to IMDb.

## What I learned

- JavaScript DOM manipulation
- Fetch API
- Working with JSON data
- API integration
- Event handling
- Search functionality
- Multi-page navigation
- User authentication basics

## Project Overview

This project contains:

- Movie search page
- Movie details page
- Login page
- Signup page
- Search functionality using OMDb API

## Technologies Used

- HTML5
- CSS3
- JavaScript
- OMDb API

## Learning Outcome

Through this project, I improved my understanding of JavaScript fundamentals, API handling, and dynamic content rendering.
