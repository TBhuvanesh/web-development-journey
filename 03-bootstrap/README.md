# Career Guidance Website

This project was built while learning **Bootstrap 5** and responsive web design.

It is a multi-page career guidance website designed to help students explore career options, learning roadmaps, and useful resources.

## What I learned

- Bootstrap grid system
- Responsive navbar
- Cards and buttons
- Accordions
- Multi-page website structure
- Utility classes
- Layout spacing and alignment

## Project Overview

This project contains:

- Home page
- Career exploration page
- Career roadmaps page
- Resources page
- FAQ page

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5

## Learning Outcome

Through this project, I improved my understanding of Bootstrap components and how to build responsive multi-page websites quickly.
