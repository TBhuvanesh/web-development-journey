# IPL Website Clone

This project was built while learning **CSS Flexbox** and layout structuring.

I recreated parts of the IPL website interface to practice real-world layout design.

## What I learned

- Flexbox alignment
- Horizontal and vertical layouts
- Navbar structuring
- Sticky positioning
- Spacing and element arrangement
- Using icons with Font Awesome

## Project Overview

This project contains:

- Header section
- Navigation bar
- Social media icons
- Match sections
- Team sections

## Technologies Used

- HTML5
- CSS3
- Flexbox

## Learning Outcome

Through this project, I improved my understanding of Flexbox and how modern websites organize content using flexible layouts.
