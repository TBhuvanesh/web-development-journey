# HTML Resume Project

This is my first project in my web development journey.

I built this resume webpage while learning the fundamentals of **HTML**.

## What I learned

* HTML document structure
* Semantic tags
* Headings, paragraphs, and lists
* Links and images
* Basic content organization
* Creating a structured resume layout

## Project Overview

This project is a simple personal resume webpage that contains:

* Personal details
* Professional summary
* Skills
* Education
* Projects
* Certifications

The goal of this project was to understand how web pages are structured using pure HTML.

## Technologies Used

* HTML5

## Learning Outcome

Through this project, I understood how HTML acts as the foundation of web development and how content is organized before styling.
